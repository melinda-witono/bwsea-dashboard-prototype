# Testing Instructions for Assignment 1

This document provides instructions on how to test your program for **Assignment 1**.

## How to Run the Test
windows:        type t1.txt | python <your_program.py>
mac:            cat t1.txt | python3 <your_program.py>

Note: If you encounter an EOF (End-of-File) related error, it might be due to incompatible line endings between different operating systems. You can try switching between LF (Line Feed) and CRLF (Carriage Return + Line Feed) formats for the text file t1.txt to resolve the issue.
